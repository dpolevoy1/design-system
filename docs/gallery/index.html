Assemble design system Internal

Shared tokens and React primitives for A.Team Assemble — color variables aligned with Figma, plus Tooltip, ToggleIndicator, and Spinner. This gallery is a static preview: run a local server from the repository (see Installation) so CSS token paths resolve.

GitHub

Installation

Consume the package from GitHub; import tokens once in your app shell.

package.json & CSS

Add the dependency and import the palette in your global stylesheet (e.g. index.css).

@assemble/design-system 
{
  "dependencies": {
    "@assemble/design-system": "github:dpolevoy1/design-system#main"
  }
}

/* app shell */
@import "@assemble/design-system/tokens/color-palette.css";

View this gallery: from the repository root, run python3 -m http.server 5175 then open http://localhost:5175/docs/gallery/ (SidebarMenu may use port 5174).

Tooltip

Default panel: dark background (--color-tooltip-panel-background), primary line in Neutral 100, shortcut in Secondary 300. React component positions the panel to the right and slightly below the pointer.

Tooltip example

Demo control

Collapse sidebar

⌘S

Static mock — in-app, wrap controls with the Tooltip component from @assemble/design-system. 

import { Tooltip } from "@assemble/design-system";

<Tooltip label="Collapse sidebar" shortcut="⌘S">
  <button type="button" aria-label="Collapse sidebar">…</button>
</Tooltip>

Toggle

Non-interactive ToggleIndicator in React; parent row supplies ARIA and toggleRowDataProps(). Full interaction rules: docs/TOGGLE.md.

Toggle example

Dark mode (off)
Dark mode (on)

Click a switch to toggle — mirrors transition tokens from Figma (8096:64).

import {
  ToggleIndicator,
  toggleRowDataProps,
} from "@assemble/design-system";

<button
  type="button"
  role="menuitemcheckbox"
  aria-checked={on}
  {...toggleRowDataProps()}
>
  <span>Dark mode</span>
  <ToggleIndicator on={on} aria-hidden />
</button>

Spinner

Indeterminate line arc, 0.9s linear rotation. Use variant primary (--color-primary-600) on light surfaces and neutral (--color-neutral-100) on dark or brand gradients.
